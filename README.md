오늘 경기도 예측해축?
====================
해외축구 경기 인터페이스 및 승률 예측 프로그램
------------------------------------------
## 주요 기능
#### 1. 로그인 기능 (아이디/비밀번호 조회시 사용자 정보 이메일로 전송)
#### 2. 당일 주요 경기 및 순위 조회 (네이버 스포츠 해외축구 탭 크롤링)
#### 3. csv파일 기반 양팀 간 승률 분석 (로지스틱 회귀 분석)
